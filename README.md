This project is a Healthcare Recommendation System built using Machine Learning. The main aim of this project is to suggest similar diseases based on symptoms. The dataset used contains different diseases along with their symptoms in the form of binary values (0 and 1). Using this data, a feature matrix is created and a Nearest Neighbors model with cosine similarity is applied to find diseases that have similar symptom patterns.

The user can select a disease from a dropdown in the Streamlit interface, and the system will display a list of similar diseases as recommendations. This project is implemented using Python, Pandas, Scikit-learn, and Streamlit for the user interface. Since the dataset is large, it is not uploaded in this repository, but it can be downloaded from Kaggle and placed in the project folder as data.csv.

To run the project, install the required libraries and execute the command streamlit run app.py. This project is created for learning purposes and is not intended for real medical use.
